## 🚀 Key Features

* **Friendship Health Dashboard:** Visualize who you're on track with and who is "overdue" for a chat using dynamic status-colored cards.
* **One-Click Check-ins:** Log calls, texts, or video chats instantly from the Friend Details page with real-time timeline updates.
* **Friendship Analytics:** Interactive data visualization using **Recharts** to analyze your communication patterns (Call vs. Text vs. Video).
* **Persistent Timeline:** A filterable history of all your interactions, ensuring you always know the last time you reached out.

## 📋 Project Structure
The application consumes a local `friends.json` database. Each profile tracks contact frequency, relationship goals, and history.
